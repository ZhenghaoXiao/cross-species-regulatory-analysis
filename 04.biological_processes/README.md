# Biological Process Enrichment

This folder contains Task 3 analysis code for linking adrenal open chromatin peak sets to biological processes.

Main scripts:

- `run_rgreat.R`: runs rGREAT enrichment for human-specific, human-shared, mouse-specific, mouse-shared, all-human, and all-mouse peak sets.
- `top10_GO_BP_Plot.py`: plots top biological process enrichments from the filtered rGREAT output tables.
- `run.sbatch`: Slurm launcher for the rGREAT step.
- `run_plot.sbatch`: Slurm launcher for the plotting step.

Inputs are generated by Task 2 in `03.mapping/`. Human-specific and human-shared analyses use recovered human-coordinate peak sets, while mouse-specific and mouse-shared analyses use native mouse-coordinate peak sets.

Results are written to `results/`.
